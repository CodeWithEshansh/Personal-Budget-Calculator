# 💰 Personal Budget Calculator

A modular **Personal Budget Calculator** built using **Python, OOP, File Handling, and Tkinter GUI** to help users track expenses, manage budgets, and improve financial planning.

---

## 🚀 Features

* ✅ Structured expense validation system
* 📊 Automated monthly expense calculations
* 💡 Category-wise budget tracking
* 🎯 Savings goal management
* 🔍 Regex-based expense search
* 💾 Persistent storage using JSON
* 🖥️ Interactive Tkinter GUI

---

## 🧠 Project Highlights

* Implemented a **modular object-oriented architecture** for scalability
* Designed a **robust validation system** for accurate budgeting
* Enabled **automated financial insights** (total expenses, remaining budget)
* Built a **user-friendly GUI** for real-time interaction

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts:** Object-Oriented Programming (OOP)
* **Storage:** JSON File Handling
* **Interface:** Tkinter (GUI)
* **Extras:** Regex for pattern-based search
